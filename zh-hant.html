什麼是 Caddy？

Caddy服務器是一個開源的，使用 Golang 編寫，支持 HTTP/2 的 Web 服務端。它使用 Golang 標準庫提供 HTTP 功能。默認啟用 HTTPS。Caddy 是第一個無需額外配置即可提供 HTTPS 特性的 Web 服務器。(Caddy安裝教程)

$ caddy file-server --domain example.com //Public file server over HTTPS
$ caddy reverse-proxy --from example.com --to localhost:9000 //HTTPS reverse proxy
$ caddy run //Run server with Caddyfile in working directory

Caddy 用戶社區推薦使用 Debian 11 + Caddy 2.0 + MariaDB 10.5.4 + PHP 8.2

CaddyServer 的特點

  • 充分的安全性保障
  • 開源，可獲取所有源代碼
  • 免費，無需高昂的商業授權
  • 配置極其簡單
  • 唯一一個默認使用 https 的 WEB 服務器
  • 支持動態網站
  • 自動配置 TLS，自動更新證書
  • 支持多核
  • 豐富的插件擴展
