什么是 Caddy？

Caddy服务器是一个开源的，使用 Golang 编写，支持 HTTP/2 的 Web 服务端。它使用 Golang 标准库提供 HTTP 功能。默认启用 HTTPS。Caddy 是第一个无需额外配置即可提供 HTTPS 特性的 Web 服务器。(Caddy安装教程)

$ caddy file-server --domain example.com //Public file server over HTTPS
$ caddy reverse-proxy --from example.com --to localhost:9000 //HTTPS reverse proxy
$ caddy run //Run server with Caddyfile in working directory

Caddy 用户社区推荐使用 Debian 10.7 + Caddy 2.0 + MariaDB 10.5.4 + PHP 7.4

CaddyServer 的特点

  • 充分的安全性保障
  • 开源，可获取所有源代码
  • 免费，无需高昂的商业授权
  • 配置极其简单
  • 唯一一个默认使用 https 的 WEB 服务器
  • 支持动态网站
  • 自动配置 TLS，自动更新证书
  • 支持多核
  • 丰富的插件扩展
